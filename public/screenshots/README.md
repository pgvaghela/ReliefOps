# Screenshots for Case Study

Place your screenshot images in this directory with the following names:

- `overview.png` - Overview Dashboard screenshot
- `shelters.png` - Shelters List screenshot  
- `alert-drawer.png` - Alert Drawer screenshot
- `incident-detail.png` - Incident Detail screenshot

The images will automatically display on the `/case-study` page. If an image is missing, a placeholder will be shown instead.

**Recommended image format:**
- PNG or JPG format
- Recommended width: 1200-1600px
- Maintain aspect ratio (screenshots will scale automatically)
